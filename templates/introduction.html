About Me

Flask와 Blueprint를 활용한 웹 개발부터 머신러닝과 딥러닝까지, 데이터 중심의 애플리케이션을 구축하는 개발자입니다.

Python과 MySQL을 기반으로 API 설계 및 데이터 서버 운영을 경험했으며, Ubuntu 가상환경에서 웹 서버와 데이터 서버를 구축하고, IPTime 포트 포워딩과 Cloudflare를 활용한 도메인 연결까지 직접 구현할 수 있습니다.

기술뿐만 아니라, 원활한 소통과 협력을 중요하게 생각합니다. 팀원들과의 적극적인 커뮤니케이션을 통해 문제를 해결하고, 개발자뿐만 아니라 기획자, 디자이너와도 원활하게 협업하며 최적의 솔루션을 도출하는 것을 목표로 합니다. 특히, 복잡한 기술 개념을 쉽게 설명하고 의견을 조율하는 과정에서 강점을 발휘합니다.

웹과 AI를 결합한 솔루션 개발에 관심이 많으며, 사용자 중심의 서비스와 데이터 기반의 인사이트를 제공하는 프로젝트를 진행하는 것을 즐깁니다.

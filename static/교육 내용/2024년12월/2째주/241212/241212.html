argmax(), argmin() 메서드¶

  • 배열에서 가장 큰 값 또는 가장 작은 값의 위치(인덱스)를 찾아준다.
    • argmax(): 가장 큰 값의 위치(인덱스) 찾기
    • argmin(): 가장 작은 값의 위치(인덱스) 찾기
  • 기본적으로 배열을 평탄화(flatten)하여 인덱스를 변환

넘파이 정렬¶

  • 정렬은 배열 요소를 순서대로 배열하는 것을 말한다.
  • 정련된 순서는 숫자 또는 문자와 같은 요소를 오름차순 또는 내림차순 순서로 배열하는 것을 말한다.
  • 넘파이 배열 객체에는 배열을 정렬하는 sort()라는 메서드가 있다.

np.sort, np.argsort() 메서드¶

  • sort() 메서드 및 np.argsort() 메서드는 배열 데이터를 정렬하는데 매우 유용
    • np.sort(arr): 원본 데이터는 그대로 두고 임시로 정렬된 배열을 보여준다.
    • 벼열명.sort(): 배열에 저장된 데이터를 정렬한 다음 정렬된 데이터를 저장
    • np.argsort(): 데이터가 정렬되었을 때의 각 데이터 위치(인덱스)를 반환

import numpy as np

# 학생 이름과 성적
students  = np.array(['Alice', 'Bob', 'Charlie', 'David', 'Eva'])
scores = np.array([88, 92, 75, 95, 85])

# 성적 정렬 및 원래 인덱스 찾기
sorted_indices = np.argsort(scores)  # 정렬하기 전 성적에서의 위치값을 저장
print(sorted_indices)
sorted_scores = np.sort(scores) # 성적 정렬
print(sorted_scores)
sorted_students = students[sorted_indices] # 위치값 저장한걸 활용하여 학생 명단 정렬
print(sorted_students)

# 결과 출력
print("학생 성적 정렬 결과:")
for i in range(len(sorted_scores)):
    print(f"{sorted_students[i]}: {sorted_scores[i]}점 (원래 인덱스 {sorted_indices[i]})")

HTML
- 가장 마지막에 잇는것이 적용
- text-shadow: 4px 4px 0px -> 아래 위 흩뿌리기
- text-indent: 들여쓰기
- .abc 클래스 , #abc id, 
- id, name의 차이
- div : 영역 설정
- padding : 영역 안쪽 여백
- margin : 영역 바깥쪽 여백
	 4개가 순서대로 써잇을경우 20px, 30px, 10px, 5px -> top, right, bottom, left
	 2개가 순서대로 써잇을경우 20px, 30px->top/bottom, right/left 
- 주석처리 : 앞쪽에 <!-- 뒤에 -->
- target : 속성에 아무런 값을 입력하지 않으면 열려있는 창에서 열림
            속성에 아무거나 값을 입력하면 새로운 창에서 열림
- form 태그 : 데이터베이스에 값을 입력할 수 있는 도구      <form name="asd" method="post or get" action="경로">
  method : get방식(주소창에 보이게 넘기는 방식), post방식(서버에서만 받을 수 있음)이 있음
  action : 경로를 지정해주지 않으면 자기 주소로 다시 돌아옴
  placeholder : 입력박스 안에 글자표시 기능
  hidden : 어디팀에서 넘겨받는지 확인하기 위한 기능, 정보도 같이 넘길 수 있음
  checkbox : 이름이 같아야함, 복수선택 가능
  radio : 이름이 같아야함, 복수선택 불가능
  button : type에 아무것도 입력 안할시 submit과 같은 기능
  select : 복수선택 가능하게 할 수 있음
  number : input 타입중 하나로 숫자를 올리거나 내리는 기능을 할 수 있음
  required : 값이 꼭 입력되어야 제출가능
  selector:?
  textarea
